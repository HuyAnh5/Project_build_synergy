Act Map Prototype

Layout giống tinh thần STS hơn: nhiều route, split / merge sạch, không crossing, và cho phép backtrack sau khi đã đi qua node. Map base giữ mức 20-30 intermediate nodes, rồi mới thêm shop / forge nhánh phụ. Combat chưa đánh thì enemy vẫn còn chờ tại đó.

Generation Rules
• Base map: 20-30 intermediate nodes, chưa tính Start / Boss.
• Sau đó mới thêm 1 Shop nhánh phụ đầu act và 1 Forge nhánh phụ gần cuối act. Cả hai luôn nằm ở row kế tiếp so với node cha.
• Rest: 2 node nếu map nhỏ hơn 25 intermediate nodes, và 3 node nếu map có từ 25 intermediate nodes trở lên.
• Hai Rest phải có ít nhất 2 node nằm giữa trên đường ngắn nhất.
• Rest row gap tối thiểu là 2 để map lớn có thể xuất hiện đủ 3 Rest.
• Event target ≈ 23% số node tồn tại, cho phép lệch khoảng ±1 đến ±2 node.
• Những node nối trực tiếp với Start có thể là Combat hoặc Event, nhưng tối đa chỉ 2 Event.
• Extra hint source xuất hiện trên Event và không xuất hiện ở lớp node đầu tiên nối với Start.
• Shop vẫn có Buy Hint riêng; ngoài ra các Hint Event phải có ít nhất 1 node nằm giữa trên đường ngắn nhất.
• Cho phép tối đa 2 Event nối trực tiếp với nhau; không tạo cụm 3 Event liền nhau.
• Mỗi node nối tối đa 4 node tổng cộng.
• Single-path tuyến tính dài tối đa 5 node; 4 node vẫn được phép.
Movement Rules
• Đi tới node kề nhau được highlight.
• Đi lên là chọn route mới.
• Đi xuống chỉ quay lại những node đã từng vào.
• Node đã clear sẽ chỉ còn vòng tròn rỗng.
• Đoạn single-path dài tối đa 5 node; 4 node vẫn hợp lệ.
